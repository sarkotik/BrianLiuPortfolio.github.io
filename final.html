Final Project

Project Description

This project essentially compelled us to apply all the concepts awe have learned this semester into one game. This taught me a lot on how to correctly format certain codes in order to take place at a specific point in time.

Game Description

The purpose of this game is to direct a person to reach the portals in order to continue to the next level, while avoiding certain obstacles and enemies. The user will control the main character through the up right and left keys in order to move horizontally as well as make the character jump. They will attempt to maneuver around obstacles and enemies to reach the end portal, which will direct them to the next level.
