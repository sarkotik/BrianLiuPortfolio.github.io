Scratch Page

Name Assignment

Description

Use scratch to design and create a program that shows my name. The requirements were to add a sprite for each letter in my name and give them three behaviors. When the green flag was clicked, it would start, and for extra credit, I added a function to reset the scene to the beginning each time the green flag was clicked.

Concepts

  • Using event blocks - green flag clicked, when key pressed
  • Conditional statements - used if statements
  • Used loops - forever blocks and repeat until blocks
  • Sprites - created my own sprites using the sprite editor. Used library of sprites to find letters of my name and adjusted costumes.
  • Movement - Used x and y values to make my sprites move to their original location when the green flag was clicked.
